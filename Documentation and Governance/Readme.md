## README
This directory serves as a container for Communitycoin documents and statements. 
This Readme serves as an index

Only statements that are explicitly endorsed by all participating Communitycoin teams have a strict editorial process as described in
[Editorial Process for Rooty Documents](/Communitycoins.Rooty/README/Documentation%20and%20Governance/Editorial%20Process%20for%20Rooty%20Documents.md). In particular they concern information presented on the [website of Communitycoins](https://communitycoins.org)

The main texts and their revisions are maintained on a main-level project: [Communitycoins master texts](https://git.communitycoins.org/Communitycoins.Rooty/-/projects/6)

- ["what are communitycoins"](/Communitycoins.Rooty/README/Documentation%20and%20Governance/) This burning question is the first example of a statement that follows this process.
- [The Manifest](https://git.communitycoins.org/Communitycoins.Rooty/README/src/branch/main/Documentation%20and%20Governance/Manifest) is the he main text on the website http://communitycoins.org is, which is signed/endorsed by all cooperating teams. Changes should be carefully communicated.
